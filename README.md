## 🧠 What is VisualGuide™?

VisualGuide™ is a **visual clarity and focus overlay tool** designed to reduce clutter and improve awareness on screen.

It provides **optional visual overlays** that help users better understand what they are seeing — without taking control away from the user.

---

## 🔍 What VisualGuide Does

- Displays **optional visual overlays** to improve clarity and focus
- Provides **highlighting and guidance elements** to improve awareness
- Reduces distractions by simplifying visual information
- Supports accessibility-focused modes like contrast and focus adjustments
- Works across games and everyday applications

---

## ⚙️ How VisualGuide Works

VisualGuide analyzes what is on your screen and provides **optional visual overlays** to assist with perception and awareness.

✔️ External overlay only  
✔️ Read-only visual assistance  
✔️ No injection into games  
✔️ No reading or writing game memory  
✔️ No mouse or keyboard automation  

All actions remain **fully controlled by the user**.
